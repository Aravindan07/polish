Polish UI is a collection of pre-built, fully responsive Components you can drop into your Front End projects. Get started by checking out our free preview components, or browsing the PNG previews in the categories you're most curious about.

Get Started

To quickly add polish ui into your project, add the below line into your HTML boilerplate code. 


						
<!DOCTYPE html>
<html>
<head>
 <!-- Required meta tags -->
<meta charset="UTF-8" />
<meta http-equiv="X-UA-Compatible" content="IE=edge" />
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
 <!-- Polish CSS -->
<link rel="stylesheet" href="https://www.polish-ui.netlify.app/index.css" />
</head>
<body>
 <!-- Optional Javascript -->
<script src="https://www.polish-ui.netlify.app/index.js"></script>
</body>
</html>

And Voila! now you have the power of Polish UI in your project.
